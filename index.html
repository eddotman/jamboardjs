Try it:

Setup:

Important Note: timbre.js and JQuery are required for jamboardJS.


Example: Setting up a jamboard mapped to the keyboard.

HTML:

<script src="js/timbre.js"></script>
<script src="js/jquery-1.9.1.js"></script>
<script src="js/jamboard.js"></script>
<div class="jamboard jmb-key"></div>

JS:

$(document).ready(function() {
var jmb1 = new Jamboard();
jmb1.genGrid();
});
